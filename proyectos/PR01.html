PR01 - De la teoría a la práctica: Walkthrough en acción

Objetivo del Proyecto

Demostrar la capacidad de aplicar conocimientos teóricos de seguridad informática en un entorno práctico mediante la realización de un walkthrough detallado. Durante el proceso se analizan vulnerabilidades en una máquina virtual vulnerable, se identifican posibles vectores de ataque y se aplican técnicas de explotación para obtener acceso al sistema.

Descripción

El proyecto consiste en la preparación de un entorno de laboratorio utilizando máquinas virtuales para realizar pruebas de seguridad. Posteriormente se ejecutan técnicas de reconocimiento, enumeración de servicios y análisis de vulnerabilidades. Una vez identificadas las posibles debilidades del sistema, se aplican exploits adecuados para obtener acceso al sistema objetivo. Finalmente se documenta todo el proceso y se realiza una demostración mediante un video walkthrough.
