PR01: De la teoría a la práctica - Walkthrough en acción

Materia: Seguridad Informática | Semestre: 8º | Fecha: 2026 | Nombre: América Fabiola Guerra Ramírez - 179884 | Profesor: Servando López Contreras

Objetivo

Demostrar la capacidad de aplicar conocimientos teóricos de seguridad informática en un entorno práctico mediante la realización de un walkthrough detallado. Durante el proceso se analizan vulnerabilidades de una máquina virtual vulnerable, se identifican posibles vectores de ataque y se aplican técnicas de explotación hasta obtener acceso al sistema objetivo.

Descripción del Proyecto

El proyecto consiste en preparar un entorno de laboratorio utilizando máquinas virtuales vulnerables. Posteriormente se realiza un proceso de reconocimiento del sistema, enumeración de servicios, análisis de vulnerabilidades y explotación de las debilidades identificadas. Finalmente se documenta todo el proceso realizado para demostrar el impacto de las vulnerabilidades encontradas y el control obtenido sobre el sistema.
